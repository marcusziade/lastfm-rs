Quick Start

Install CLI

# Install from source
cargo install --path . --bin lastfm-cli

# Or use the install script
./scripts/install-cli.sh

Authenticate

# Login to Last.fm
lastfm-cli auth login

# Check status
lastfm-cli auth status
